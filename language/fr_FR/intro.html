Fsrmp signifie : filter system recently modified pictures. filtre les images récemment modifiées sur le système.

Quand de nombreuses images sont mises à jour par une autre application dans des catégories éparses, il peut être utile de les manipuler par le biais de la gestion par lot. Ce plugin fourni une paire de filtres paramétrables pour récupérer les fichiers images récemment modifiés sur le système.

Exemple: considérons que de temps en temps, vous ajoutiez des étiquettes à vos photos avec le logiciel jBrout (ou digikam, ...). Vous serez content que PIWIGO puisse rapidement synchroniser les méta-données de ces photos.
